== Description ==
Temporarily stopped !!!!!!!!!! you might also try <a href="https://wordpress.org/plugins/add-hierarchy-parent-to-post/">Add hierarchy (parent) to posts</a> plugin


You can have custom-post-type URL's without their 'slug' in the front (for example, 'product' type url will no longer be: example.com/product/ball  , but becomes: example.com/ball.   (Useful when you dont want to use default posts)
